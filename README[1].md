# ❤️ Heart Disease Prediction Model

## 📌 Objective
To predict whether a person is likely to have heart disease using Machine Learning.

## 🚀 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Flask (for deployment)

## 📊 Dataset
UCI Heart Disease Dataset

## ⚙️ How to Run
1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Train model:
   ```bash
   python model.py
   ```
3. Run app:
   ```bash
   python app.py
   ```

## 📈 Output
Predicts 0 (No Disease) or 1 (Disease)
